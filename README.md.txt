Como rodar a aplicação: 
• Deve ser feito o download do Visual Studio Installer;
• Após abrir o Installer, deve ser instalado o Visual Studio Community 2019 junto com algumas cargas de trabalho( Modificações );
• Elas são: "Desenvolvimento para desktop com .NET" e "Processamento e armazenamentos de dados";
• Após baixar e importar o repósitorio, deve ser inserido a conexão de dados;
• Deve ser aberto a Janel de Gerenciador de Servidores - Conexões de dados - BDGERAL, após apertar nele deve ser visualizado nas propriedades a cadeia de conexão e ser colada a onde está indicando no Program.cs;
• Após isso rode o programa.


Tecnologias
• Visual Studio 2019
• Banco de Dados baseado em Serviço
• Aplicativo do Console (.NET Framework)
